README
======

This is the front end for VSearch.

## Running and building

From the root folder:
* `ionic serve` Runs the angular development server.
`ionic build` Transpiles and copies all files to the www folder. This can be zipped and dropped into a web server manually or through an automated devops process.

Stack: Ionic 3.9.9 + Angular 5.2.11
